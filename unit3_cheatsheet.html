THE Ruby Basics Cheat Sheet

Ruby is a great programming language because takes advantage of progression in hardware to make the life of programmers easier by abstracting away a lot of annoying details (like garbage collection) that slow down programmers down. Here is a collection of basic commands to get started using Ruby:

How to make a function:

def function_name

THE FUNCTION CODE

end

___

How to create an array:

new_array = [1,2,3]

That's it! We did a lot in that one line. We implicitly declared a variable called new_array, and we assigned it an array with three elements - [1,2,3]. Note that the array's type declaration is also implicit. Ruby is smart enough to avoid boggling programmers down with type declaration boilerplate.

___

How to create a hash :

new_hash = {:first_key => 'first_value', :second_key => 'second_value'}

This can also be rewritten as:

new_hash = {first_key: 'first_value', second_key: 'second_value'}

Using first_key: is functionally equivalent to :first_key =>. It is a shorthand for both declaring a symbol and assigning that symbol to a value.

___
